Relays

"actual" is the state reported by the GPIO relay manager. LN₂ open prompts a confirmation since it physically actuates the dewar transfer valve. The pressure-sensor relay enables the FRG vacuum gauge controller power. Sparklines show ~30 min of recent values (cleared on page reload — open Grafana for history).

heater range
Output 1 ranges step in factor-of-10 power increments (5 ≈ 50 W). Heater is fully off when range = 0, regardless of output mode.
output mode
Closed Loop = PID controls heater from setpoint. Open Loop = manual output only (set with the field below). Off = no heater current.
manual output %
Only effective when output mode is Open Loop / Closed Loop / Zone. In Closed Loop this gets summed with the PID output.
setpoint ramp rate
When on, the controller walks toward the setpoint at the configured rate (0.1–100 K/min) instead of jumping. "ramping" indicator above shows whether a ramp is in progress.
